
프롬프트 2: "항원 검사" 페이지 신규 제작 요청**

## 1. 접근 경로

* 의사용 Navigation Bar → **항원 검사**

## 2. 항원 검사 페이지 기본 요구사항

* 상단에 **‘최종 처방전 생성’ 버튼**이 필요하다.
  (항원 검사 결과를 기반으로 최종 처방전을 만들기 위함)

## 3. 환자 선택 기능

* 항원 검사 페이지에서 **환자를 선택**하면
  해당 환자의 **검사 결과 입력 페이지**로 이동한다.

## 4. 검사 결과 입력 페이지 기능

### 검사 결과 추가 방식

* ‘+’ 버튼으로 입력 필드를 여러 개 추가할 수 있다.
* 각 입력 항목은 다음 구조를 가진다:

```
{
  type: 'DNA' | 'RNA' | 'PROTEIN',  // 3가지 중 하나 선택
  value: string                      // 문자열 입력
}
```

### 예측 요청

* ‘예측’ 버튼을 누르면 Flask 서버로 검사 결과 전체를 전송한다.
* Flask는 아래 3가지 예측 결과를 반환한다:

  1. task1: 정상 / 항원 분류
  2. task2: 정상, 감기, 독감, 코로나 분류
  3. task3: 30종류의 단백질 중 하나로 분류

## 5. 예측 결과 출력 형식

예측 결과는 아래 포맷의 리스트로 화면에 표시한다:

1. 인덱스 번호
2. 입력 문자열
3. task1 결과(정상 / 항원)
4. task2 결과(정상·감기·독감·코로나)
5. task3 결과(단백질 종류)
6. **더보기 버튼(팝업 카드)**

### 범주 정렬 규칙

정렬 순서:
**코로나 → 독감 → 감기 → 정상**

### 색상 규칙

각 범주는 **우선순위에 따라 색상이 확실히 구분되도록 적용**해야 한다.

## 6. 더보기 버튼 기능

* 더보기 버튼 클릭 시 팝업 카드로 다음 정보를 표시한다:

  * 단백질의 3D 시각화
  * 해당 단백질의 특성 설명
