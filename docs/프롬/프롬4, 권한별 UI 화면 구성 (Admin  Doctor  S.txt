프롬4, 권한별 UI 화면 구성 (Admin / Doctor / Staff)

아래는 **Admin / Doctor / Staff(간호사·일반 직원)** 3가지 역할에 대해
**각 권한별로 명확히 차별화된 화면 구성(UI 구조)**을 정리한 문서이다.
*요구사항: “doc 폴더의 문서 참고해서 역할별 필요한 화면을 분리하여 구성


# 1) 🟥 **Admin 화면 구성 (최고 관리자)**

*권한 수준: 시스템 전체를 관리하는 최고 권한.*

## 📌 메인 기능

### ① 대시보드 (시스템 운영 모니터링)

* 전체 사용자 수, 의사 수, 환자 수
* 오늘 발생한 Encounter 개수
* 오류 로그, 서버 상태
* AI 예측 요청 수 / 성공률 통계

### ② 사용자 관리(User Management)

* 사용자 목록(검색/필터)
* 역할(Doctor / Staff / Patient) 변경
* 계정 활성/비활성
* **의사 등록/승인** 기능

### ③ 권한/그룹 관리 (Role & Permission)

* Django Group(Role) 생성/수정
* 권한(permission) 추가
* 특정 유저에게 개별 권한 부여

### ④ 병원 조직 관리

* 진료과 관리 (정신건강의학과, 신경과 등)
* 의료진 소속 관리
* 직원근무 스케줄 관리(옵션)

### ⑤ AI 모델/서버 관리

* 모델 버전 업로드
* 모델 상태 확인
* AI 로그 확인
* XAI 저장소 관리

### ⑥ 시스템 설정

* 환경 설정(.env 참고: API endpoint)
* 파일 보안 설정
* Audit Log 확인

---

# 2) 🟦 **Doctor 화면 구성 (주요 사용자)**

*권한 수준: 환자 진료 및 AI 결과 확인 중심*

## 📌 메인 기능

### ① 오늘의 환자 목록 (Doctor Dashboard)

* 오늘 예약된 환자 리스트
* 환자별 상태 요약
* 최근 Encounter 바로 열기

### ② 환자 검색/조회 (Patient Browser)

* 이름/생년월일/ID로 검색
* 나에게 배정된 환자 목록(주치의 환자 리스트)
* 환자 기본 정보 + 요약 그래프(최근 AI 위험도, 최근 vitals)

### ③ Encounter(진료) 화면

* **3-pane 구조(OHIF 참고)**

  * 왼쪽: 접수 정보, 과거 진료 기록
  * 중앙: 영상 뷰어(CT/MRI) + AI segmentation overlay
  * 오른쪽: SOAP / Vitals / AI 분석 결과

### ④ EMR 기록 작성

* SOAP 작성
* 처방 입력(Prescription Create)
* 진단 코드 선택
* 자동 요약(AI 기반) 버튼

### ⑤ AI CDSS (XAI 기반)

* AI 예측 결과(진단/확률)
* SHAP / Grad-CAM 시각화
* Merged Document preview
* 의사 피드백 입력

### ⑥ 스케줄 / 예약 관리

* My Schedule
* 예약 확인 & 변경

---

# 3) 🟩 **Staff 화면 구성 (간호사/원무/일반 스태프)**

*권한 수준: 환자 관리 및 행정 처리 중심. 의학적 의사결정 권한 없음.*

## 📌 메인 기능

### ① 환자 등록 / 기본 정보 수정

* 신규 환자 등록
* 주소/전화번호 등 기초 정보 수정
* 보험 정보 입력

### ② 예약 접수 기능

* 환자 예약 생성/수정/취소
* 의사별 스케줄 보기
* “대기 환자 보기(Waiting room)”

### ③ 간호 기록 입력 (Nursing Note)

* 기본 Vitals 측정 입력
* 체온, 맥박, 혈압 입력
* 자동 그래프 생성

### ④ 문서/서류 관리

* 진단서 요청 접수
* 입/퇴원 서류 관리(옵션)
* 추가 증빙 파일 업로드

### ⑤ 알림(Notifications)

* 환자 호출(“OOO 님 들어오세요”)
* 접수 완료 알림

---

# 🔥 역할별 차별화 포인트 정리

| 역할         | 핵심 목적      | 핵심 화면                          |
| ---------- | ---------- | ------------------------------ |
| **Admin**  | 시스템 관리     | 사용자/권한 관리, AI 모델 관리, 병원조직 관리   |
| **Doctor** | 진료 + AI 해석 | 환자/Encounter, 영상 뷰어, AI XAI 화면 |
| **Staff**  | 접수/행정/간호   | 예약, 환자 정보, Vitals 기록           |
