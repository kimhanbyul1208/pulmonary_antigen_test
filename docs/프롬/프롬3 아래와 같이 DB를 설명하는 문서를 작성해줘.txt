# 각 테이블의 내용은 따로 설명

| 테이블                            | 무엇을 저장?      | 요약        |
| ------------------------------ | ------------ | --------- |
| **auth_permission**            | 권한 정의 목록     | 권한 카탈로그   |
| **auth_group_permissions**     | 그룹(role)별 권한 | 역할에 권한 매핑 |
| **auth_user_user_permissions** | 개별 유저 추가 권한  | 예외적 권한 부여 |
| **django_session**             | 로그인 세션       | 로그인 상태 유지 |
| **django_migrations**          | 마이그레이션 적용 기록 | DB 버전 관리  |
